# Hi, I'm Juan

I've been writing code since before I knew it had a name. Back then it was a private server for an online MMORPG running on whatever hardware I could get my hands on. Now it's microservices, cloud infrastructure, and CI/CD pipelines — but the feeling is the same: build something, break it, understand why, make it better.

I'm a Senior Software Engineer with 13+ years of experience. I work fully remote with distributed teams, and I care a lot about the craft: clean code, good processes, reducing friction.

Lately I've been learning Go seriously, exploring LLMs, and using AI tools to finally unblock personal projects that had been sitting in a "someday" pile for years. I write about it at [jdgomez.dev](https://jdgomez.dev).

## What you'll find here

**[civbot](https://github.com/jdgomez/civbot)** — A Go webhook receiver that bridges Civilization 6 save events to Discord notifications. Built to scratch an itch, works in production (with HTTPS).

**[DSB — Dance Sequence Builder](https://github.com/jdgomez/DSB)** — A Go CLI for building dance sequences from known figures and song durations. Work in progress, built while practising vibe coding.

**[Katas](https://github.com/jdgomez/Katas)** — Where I practice. Currently using Go to sharpen fundamentals. Each kata has its own README.

**[HuggingFaceLearn](https://github.com/jdgomez/HuggingFaceLearn)** — Notebooks from my HuggingFace LLM course. The place where I started understanding what's actually happening inside these models.

**[keyboards](https://github.com/jdgomez/keyboards)** — Config files for my hand-soldered Corne split keyboard running Dvorak. Not software engineering, but it is engineering.
